# AI Operations Dashboard

Современный дашборд для мониторинга операций с использованием AI.

## Технологии

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Recharts (для графиков)
- Lucide React (иконки)

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## Сборка

```bash
npm run build
```

## Особенности

- **Top Bar**: Логотип, глобальный поиск, уведомления, профиль, переключатель окружения
- **Sidebar**: Навигация по разделам (Overview, KPIs, Fraud, Recommender, Fleets, Settings)
- **KPI Cards**: 4 карточки с метриками и drill-down функциональностью
- **Main Charts**: Линейный график объемов/выручки/маржи с переключением периодов
- **AI Insights**: Рекомендации с возможностью применения
- **Data Table**: Таблица транзакций с фильтрами и inline expansion
- **Smart Fuel Recommender**: Плавающий виджет с топ-3 АЗС

## Размеры

- Desktop: 1440×900px
- Header: 64px высота
- Sidebar: 280px ширина
